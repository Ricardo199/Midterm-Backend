# COMP229.2024.Backend

## Project Overview

This project includes examples for the COMP229 course on backend development.

## Contents

- Week 1: Introduction
- Week 2: Setup
- Week 3: Databases
- Week 4: RESTful APIs
- Week 5: Auth
- Week 6: Error Handling
- Week 7: Deployment

## Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/COMP229.2024.Backend.git
cd COMP229.2024.Backend
```

## Prerequisites

- Node.js
- npm
- MongoDB

## Installation

Install dependencies:

```bash
npm install
```

## Running the Project

Start the project:

```bash
npm start
```
